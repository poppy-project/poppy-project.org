Obtenir de l'aide technique, signaler un bug ou émettre un souhait de fonctionnalité

C'est toujours une bonne idée de commencer par parcourir la documentation, y compris la section de résolution des problèmes. S'il s'agit d'un bug, signalez-le idéalement comme issue dans le dépôt Github concerné. Si vous ne savez pas l'identifier, utilisez le le forum.

Si vous avez encore besoin d'aide, lancez une nouvelle discussion pour obtenir de l'aide technique de la communauté. N'oubliez pas de préciser quelle est votre créature, votre version d'image SD ou de Pypot.
